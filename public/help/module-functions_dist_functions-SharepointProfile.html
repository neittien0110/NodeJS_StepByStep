SharepointProfile

Description

Thông tin profile của sinh viên, cán bộ trên hệ thống Sharepoint.

Parameters
Name Type Description
EmailSinhVien string

Email HUST của sinh viên. Ví dụ chinh.bq002987@sis.hust.edu.vn
Returns

URL Profile trên hệ thống Sharepoint trong Office365 của sinh viên đó. Lưu ý: người xem phải đăng nhập vào Office365 theo tài khoản email trường để có quyền xem thông tin tài khoản.

Examples
=SharepointProfile("long.lv202987@sis.hust.edu.vn") =https://husteduvn-my.sharepoint.com/person.aspx?user=long.lv202987@sis.hust.edu.vn
